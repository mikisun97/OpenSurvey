# TinyMCE 환경 변수 설정 가이드

## 개발 환경 (.env.local 파일 생성)
NEXT_PUBLIC_TINYMCE_API_KEY=your-api-key-here
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080/api

## 배포 환경 (Vercel Environment Variables)
NEXT_PUBLIC_TINYMCE_API_KEY=your-api-key-here
NEXT_PUBLIC_API_BASE_URL=https://opensurvey-production.up.railway.app/api

## TinyMCE API 키 발급 방법:
1. https://www.tiny.cloud/auth/signup/ 에서 무료 계정 생성
2. Dashboard → API Key Management → Create API Key
3. Approved Domains에 다음 도메인들 추가:
   - localhost:3000
   - your-app.vercel.app
   - *.vercel.app

## 주의사항:
- .env.local 파일은 Git에 커밋하지 마세요
- API 키는 공개되면 안됩니다
- Vercel 배포 시 Environment Variables에서 설정하세요
